# 短篇网文写作框架

## 写作心态

### 短篇的优势
- 完成周期短（1-7 天 vs 长篇的 6-24 个月）
- 试错成本低（写不好就换一篇）
- 即时反馈（发出去很快能看到数据）
- 可以快速积累经验

### 短篇的挑战
- 每个字都必须有用（没有水字数的空间）
- 情绪控制要求高（情绪不对全篇白费）
- 反转设计是硬功夫（不是想翻就能翻）
- 竞争更新更快（风口来得快去得快）

## 写作流程

### 第一步：定情绪

写之前先回答：**读者读完应该是什么感觉？**

情绪目标决定了：
- 选什么题材
- 用什么结构
- 怎么设计反转
- 怎么写结尾

### 第二步：定结构

根据情绪目标选择结构：

| 情绪目标 | 推荐结构 | 反转位置 |
|----------|----------|----------|
| 意难平 | 三幕反转式 | 70% |
| 震惊 | 套娃反转式 | 50% + 80% |
| 爽感 | 升级打脸式 | 每个 20% |
| 治愈 | 双时间线式 | 60% |
| 共鸣 | 现实递进式 | 无大反转 |

### 第三步：定人设

短篇人设要简洁有力：

```
主角：{一句话人设，包含一个核心矛盾}
例：「表面温顺的儿媳妇，其实一直在记录婆婆的每一句话」

关键角色：{一句话人设，和主角形成对立或互补}
例：「强势但内心孤独的婆婆」
```

短篇通常只需要 2-4 个角色，每个角色必须有明确功能。

### 第四步：定反转

在写之前就确定反转：

```
反转类型：{}
表面故事：{读者一开始以为的}
真实故事：{实际上发生的}
铺垫线索：{至少 3 个}
误导方向：{把读者往哪引}
揭示方式：{怎么揭开真相}
```

### 第五步：写初稿

按结构分段写，不要回头改：

1. **开头**：3 句话内抓住读者（500 字）
2. **铺垫**：建立人物，埋线索，推情绪（30-40% 篇幅）
3. **升级**：矛盾激化，情绪推高（20-30% 篇幅）
4. **反转**：引爆情绪炸弹（10-15% 篇幅）
5. **结尾**：情绪落定（5-10% 篇幅）

### 第六步：精修

#### 第一轮：删减
- 删掉不推动剧情的对话
- 删掉不铺垫反转的描写
- 删掉不推高情绪的心理活动
- 目标：删掉 10-20% 的字数

#### 第二轮：强化
- 检查开头 3 句话是否足够抓人
- 检查反转铺垫是否自然
- 检查情绪曲线是否平滑
- 检查结尾是否有余韵

#### 第三轮：打磨
- 优化关键段落的语言
- 检查节奏感（快慢交替）
- 确认人物言行一致
- 通读一遍感受整体效果

## 平台适配

### 知乎盐言适配要点
- 重视文笔质量（盐言读者对文字有要求）
- 重视情绪深度（不能只是爽，要有触动）
- 篇幅 1-3 万字
- 标题要有吸引力但不标题党
- 开头要像散文一样精致

### 番茄短篇适配要点
- 重视节奏（番茄读者耐心更短）
- 重视爽感（情绪冲击要直接）
- 篇幅 8000-15000 字
- 标题直白，一看就懂
- 开头直接进入冲突

### 小红书故事适配要点
- 生活感强
- 重视共鸣（读者要觉得「这说的不就是我吗」）
- 篇幅 2000-8000 字
- 标题用「提问式」或「悬念式」
- 开头用代入式提问

## 写作技巧库

### 对话技巧
- 每句对话都要有用（推进剧情/揭示性格/铺垫反转）
- 角色说话要有区分（身份、性格、情绪状态）
- 紧张时用短对话，舒缓时可以长对话
- 用对话暗示信息，而不是直接说出来

### 描写技巧
- 描写为情绪服务
- 用细节而不是形容词（「她攥紧了指甲」比「她很紧张」好）
- 关键时刻的描写要慢（时间拉伸感）
- 日常时刻的描写要快（不要拖节奏）

### 反转技巧
- 反转要在写之前就定好
- 铺垫要自然，不能刻意
- 误导要合理，不能硬拐
- 揭示要干脆，不能拖
- 留至少一条线索让读者回看时能发现

### 情绪技巧
- 情绪不要写出来，要让读者感受到
- 用行为而不是内心独白表达情绪
- 情绪转折要有铺垫，不能突然变化
- 高潮前的压抑越强，释放越爽

## 常见问题

### 开头写不出来
→ 跳过开头，先写反转段落，再回来写开头

### 反转设计不出来
→ 从结局倒推：你想让读者最后知道什么？这个信息怎么在最开始隐藏？

### 篇幅控制不住
→ 短篇不是小长篇，不能有「铺垫够了再说」的想法。每段都要有推进。

### 结尾收不住
→ 反转后 500 字内收尾。结尾不是总结，是最后的情绪一击。
