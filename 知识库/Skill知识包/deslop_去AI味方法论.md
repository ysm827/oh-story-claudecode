# 去 AI 味方法论

## 核心理念

AI 生成的文本有可识别的模式（高频词、句式、节奏）。去 AI 味不是改掉所有 AI 写的句子，而是消除那些让读者「觉得不对劲」的模式。

## AI 文本的 7 大特征

| 特征 | 表现 | 严重度 |
|------|------|--------|
| 高频词 | 「仿佛」「宛如」「不禁」「缓缓」「嘴角微微上扬」 | 高 |
| 弱化副词 | 「似乎」「好像」「或许」「大概」削弱确定性 | 高 |
| 意义膨胀 | 用华丽修辞包装空洞内容 | 中 |
| 万能结论 | 每段结尾都有总结性陈述 | 中 |
| 论文体 | 「首先…其次…最后…」的逻辑结构 | 中 |
| 书面语 | 对话中用书面语而非口语 | 低 |
| 三连排比 | 三个并列结构堆叠情绪 | 低 |

## 3-Pass 框架与 6-Gate 映射

| Pass | 目标 | 对应 Gate |
|------|------|-----------|
| Pass 1 | 扫描禁用词+句式+心理外化 | Gate A + B + C |
| Pass 2 | 深化句式+节奏调整 | Gate B 深化 + D |
| Pass 3 | 对话+结尾+最终润色 | Gate E + F |

## 6-Gate 详解

### Gate A：禁用词
替换高频 AI 词汇，用更自然、更具体的表达。

### Gate B：句式
打破 AI 偏好的对称句式、三连排比、过度从句。

### Gate C：心理外化
将内心独白转化为动作和对话，用「展示」代替「告知」。

### Gate D：节奏
消除均匀的段落长度，制造长短交替的阅读节奏。

### Gate E：对话
让对话更口语化、更有个性，每个角色有独特的说话方式。

### Gate F：结尾
消除总结性结尾，用留白、动作、场景结束章节。

## 操作原则

1. 逐句审查，不要批量替换
2. 保持原意不变，只改变表达方式
3. 每次修改后重读，确保不影响节奏
4. 优先修复高严重度问题
5. 不是所有 AI 特征都要改——以读者感受为标准
