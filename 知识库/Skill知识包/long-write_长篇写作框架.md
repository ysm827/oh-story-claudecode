# 长篇网文写作框架

## 写作心态

### 网文写作的现实
- 长篇网文平均 200-300 万字
- 日更 4000-6000 字是基本要求
- 从开书到签约平均需要 30-50 章
- 从签约到上架平均需要 80-120 章
- 大部分收入集中在前 10% 的作者

### 正确的心态
1. 网文是产品，不是艺术品（先满足读者需求）
2. 完成比完美重要（日更不能断）
3. 数据导向（订阅/追读说了算，不是自己感觉）
4. 长期主义（一本 200 万字的书要写 1-2 年）

## 大纲体系

### 三级大纲法

#### 一级：全书大纲
```
总字数：{X} 万字
总卷数：{Y} 卷
核心主线：{一句话}
结局：{主角最终状态}
```

#### 二级：卷级大纲
```
第 N 卷：{卷名}
- 字数：约 {X} 万字
- 章节数：约 {Y} 章
- 核心事件：{一句话}
- 本卷爽点：{2-3 个大爽点}
- 起始状态 → 结束状态
- 主要新角色
```

#### 三级：章级细纲（只做前 30 章）
```
第 N 章：{暂定名}
- 字数目标：{X} 字
- 核心事件：{一句话}
- 出场人物：{列表}
- 本章爽点/钩子
- 章尾悬念
- 承接上章：{上一章结尾状态}
```

### 为什么只做前 30 章细纲

1. 前 30 章决定签约和推荐
2. 30 章之后你对角色已经熟悉，不需要细纲
3. 过早规划远处细节，后面大概率要改

## 世界观构建

### 最小可行世界观

开书时你只需要确定：

1. **时代/背景**：现代/古代/架空/未来
2. **核心设定**：这本书最独特的设定（金手指/系统/超能力）
3. **力量体系**：如果有，只定大框架，细节后面补充
4. **社会规则**：影响剧情的关键规则（门派/组织/阶层）

### 世界观展开原则

- **按需展开**：写到哪展开到哪，不要一次设定完
- **冰山理论**：只给读者看 10%，你心里要有 100%
- **一致性**：已设定的不能自相矛盾
- **可扩展**：留足扩展空间

## 人物管理

### 主角卡

```
姓名：
年龄：
外貌（关键词）：{3-5 个词}
性格（关键词）：{3-5 个词}
核心动机：{他为什么做这件事}
金手指/能力：{}
口头禅/习惯：{}
弱点：{}
```

### 配角管理

| 角色 | 功能 | 出场章 | 退场/转化章 | 备注 |
|------|------|--------|-------------|------|
| {名字} | {导师/对手/盟友} | {N} | {M} | {关键转变} |

### 人物一致性检查

每写 20 章检查一次：
- 主角行为是否符合人设？
- 角色关系变化是否有铺垫？
- 有没有角色突然「消失」了？

## 写作技巧

### 章节结构

```
[开头] 承接上章悬念 / 新场景引入
[发展] 推进剧情 / 铺垫爽点
[核心] 本章主要事件 / 爽点释放
[收尾] 章尾钩子 / 新悬念
```

### 对话写作

- 对话要推进剧情或揭示性格
- 每个角色的说话方式要有区分
- 不要用对话做信息倾倒
- 紧张时刻用短对话，舒缓时刻可以长对话

### 场景描写

- 场景描写为情绪服务
- 紧张场景少描写多动作
- 日常场景可以多描写但要有人物互动
- 不要大段描写风景/建筑（除非是关键设定）

### 战斗/冲突写作

```
1. 建立对峙（双方实力展示）
2. 交锋（你来我往，有来有回）
3. 危机（主角陷入劣势）
4. 转折（金手指/策略/援军）
5. 胜利（爽点释放）
6. 收获（战利品/信息/成长）
```

## 常见问题

### 卡文怎么办
- 回看大纲，确认当前进度
- 跳到下一个爽点往前写
- 写一个配角视角的插曲
- 不要停下来「等灵感」

### 数据不好怎么办
- 分析是哪个环节出了问题
- 检查开头是否吸引人（完读率低 = 开头问题）
- 检查爽点密度（追读下降 = 爽点不够）
- 检查更新频率（断更 = 数据断崖）
- 不要轻易大改，先调整节奏

### 签约不上怎么办
- 确认题材是否有市场
- 检查前三章是否到位
- 对比同类型已签约作品
- 考虑换题材或换平台
