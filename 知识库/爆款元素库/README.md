# 爆款元素库

## 用途

收集和整理网络小说中反复验证过的爆款元素，为写作提供灵感参考。

## 分类

### 题材元素
- 经典题材：重生、穿越、系统、种田、宫斗、都市
- 新兴题材：规则怪谈、第四天灾、直播、短剧穿越
- 跨界题材：科幻+仙侠、都市+末世、悬疑+言情

### 设定元素
- 金手指类型：系统面板、随身空间、预知未来、重生记忆、超能力
- 世界观模板：学院流、宗门流、都市异能、末世生存、星际争霸

### 爽点元素
- 打脸类：碾压、反杀、身份曝光、实力展示
- 获得类：捡漏、传承、奇遇、系统奖励
- 反转类：绝境翻盘、真相大白、敌我转换
- 情感类：守护、重逢、告白、认可

### 钩子元素
- 开头钩子：死亡预告、神秘来电、意外发现、反常现象
- 章尾钩子：悬念、反转、新角色、危机、承诺

## 贡献方式

在使用过程中发现新的爆款元素，直接添加到对应分类下。
