## 开篇技巧

### 黄金一章法则

- 不再有"黄金三章"的奢侈，现在就是黄金一章
- 从故事最精彩、最有冲突的地方写起
- 主角要么身处危机命悬一线，要么处于令人羡慕的状态，要么被丢入完全陌生的环境
- 字数分配见文末"七步法第一章字数分配参考"

### 六大标准（参考进击的巨人第一集）

1. 从最吸引人的地方讲起
2. 用吸引人的方式交代背景（可以一句话不说就传达信息）
3. 主角及时出场，一开始就展示不同于常人的特质
4. 写好群像
5. 奠定全篇基调
6. 给人留下深刻印象

### 短视频化趋势

- 读者注意力集中时间下降，开头必须更紧凑
- 以前慢热型还有空间，现在越来越窄
- 硬核干货不如有趣的表达更吸引人

### 冲突节奏与背景融入技法

- **故事的本质是冲突**："文似看山不喜平"，冲突（矛盾）就是"不平"，写故事就是抓住"不平"
- **冲突不能断也不能过密**：冲突像打羽毛球，球要在空中飞（你来我往）才好看，球掉下去（冲突没了）没意思，但一直飞不停观众会疲惫
- **两个紧张故事之间要有缓和**：让读者缓一缓神经，但"冲突密度的缓和"和"故事之间的钩子"是两回事，不要混淆
- **开头的致命错误**：开头来一大段背景介绍、信息铺垫，全是"缓"没有"冲突"，读者没有耐心看完
- **正确做法：背景融入冲突**：让读者在不知不觉中陷入故事，《斗破苍穹》第一章用旁人议论萧炎的方式同时完成背景介绍和冲突建立
- **读者的耐心有限**：观众容忍不了太长时间的"缓和"，在网文里读者的耐心比电影更短，这个限度与日更模式和网站风格有关

### 开篇结构范例：鬼灭之刃拆解

**起**（建立共情）
- 风雪山林破落木屋，炭治郎冒雪卖炭（人设：明理、顾家）
- 温馨家庭日常和告别（人设：温柔，为后续共情铺垫）
- 路上自白：日子艰辛却幸福（人设：坚强乐观）
- 卖炭好人缘，用嗅觉帮人（人设：乐于助人，四板斧完成全方位人设）
- 对话引出世界观（鬼、猎鬼人），不超过三十字嵌入主线

**转**（危机与反转）
- 惨案发生，炭治郎背负唯一幸存的妹妹
- 妹妹鬼化，新的危机来临——矛盾从家庭悲剧转为兄妹与鬼的矛盾
- 义勇出现差点杀妹妹——矛盾再转为炭治郎与义勇的对立
- 炭治郎跪地求饶被斥，获得变强动力和中期目标
- 智慧反击展现战斗智商——无围观者时，敌人就是最大的震惊对象

**合**（新篇章开启）
- 义勇留下短期目标（找培育师）和中期目标（加入鬼杀队）
- 凄冷景色渲染前路坎坷，牵着唯一亲人奔向未知
- 每个出场人物都有功能，没有水分

### 开篇循环套路

- 主线式推进：杀鬼变强 -> 修养 -> 杀鬼再变强
- 过渡场景直接跳过，节奏快
- 紧扣情绪写，节奏鲜明，合理制造泪点和爽点

### 日常文/狗粮文开篇

**无事业线时怎么写**
- 抽取一段具有广泛共鸣的人生主线作为剧情主干（高中、大学最有效）
- 将感情线当做藤蔓依附在人生主线上生长
- 日常场景都能穿插男女主互动

**高好感度开局的循环套路**
- 不要把身体接触（牵手、接吻）当升级突破点——一次性的，用完就没感
- 学习升级文的循环体系，找到可重复利用的剧情架构
- 核心公式：稳定的人设 + 固定的剧情流程 = 读者稳定的期待
- 例：爱吃醋的女主 + 其他女偷家 = 读者永远爱看的"世界名画"
- 框架固定但内容不同，读者预料到了也乐此不疲

### 开篇信息团排版

- 不要试图在第一章塞入所有设定（身份、危机、女主、金手指、世界观）
- 设定展开几百字就会严重破坏观感
- 分批释放信息，优先级：危机感 > 人设 > 金手指暗示 > 世界观
- 每次释放都要服务于当前情绪目标
- 身穿开篇如果剧情紧张，信息不用给特别具体，等主角处理完事件再补

### 主角调子起太高

- 开局即巅峰会失去成长空间，冲突失去意义
- 后续很难设计合理的升级线和蜕变线
- 如果必须高起点，要设计等量级的对手或全新的挑战维度

### 设定量控制

- 背景设定：时间地点、等级体系、资源、势力、历史、地理、社会风俗
- 开篇只展露与剧情直接相关的部分
- 先写完开头再用审核标准自查，不要按标准去填空

### 跨媒介陷阱：看动漫写网文要谨慎

- 日漫圣母型主角（如鸣人嘴遁说服敌人）在网文里行不通，两种媒介的读者期待完全不同
- 动漫靠画面和声优支撑的桥段，纯文字无法传达同等冲击力
- 对标时应学习叙事结构和情绪节奏，而非照搬具体桥段

### 跟风/蹭热度开篇

- 写已经火起来的题材是现实选择
- 关键是要在同类题材中被读者记住
- 需要找到差异化的"活儿"——独特的视角、人设或金手指运用方式
- 拆书时不仅拆桥段内容，还要拆思路、想法和别人的套路

### 开篇信息差与期待感构建

**信息差拉期待的范本**（《练气千年，方知是封神！》）
- 首订3000+已有精品徽章，标准模版式循环升级文
- 核心手法：重生/穿越带来的信息差作为金手指基础
- 信息差→主角知道别人不知道的事→利用信息差解决问题→收获爽感
- 后期字数多了随便一个推荐就能赚钱——标准升级文的复利效应

**期待感公式**
- 期待感 = 情绪 + 情节（人设+动机）+ 信息差
- 三者缺一则期待感不足
- 信息差铺垫不够时：写了金手指信息但没从其他角度解释信息差，危机跟主角无关，自然没有期待感
- 短平快过渡到下一个有用剧情，继续"情报→收获→震惊"的循环

### 书名/简介/开篇三位一体

- 书名告诉读者金手指和题材（如"重生鉴宝"=重生金手指+鉴宝题材）
- 简介补全书名没说清的卖点；只写三件事——主角处境、金手指能干嘛、第一个爽点预览
- 开篇正文必须呼应书名和简介的承诺
- 三者链接紧密→筛选读者精准→留存率高
- **书名常见翻车**：三字四字看不出亮点是大忌；不要拿悬念当期待感，直接告诉能干嘛；多扫三江榜/销售榜学习同类型书名

**鉴宝都市开篇范例**（《重生鉴宝，我真没想当专家》）
- 核心结构：鉴定→发现隐藏价值→震惊围观→收获奖励→下一个鉴定
- 变着花样重复这个套路就够赚钱

### 开篇的自我审核

- 写完开头再自查，不要按标准去填空
- 过于抽象的审核标准会导致执行偏差——不由自主写成填空
- 正确做法：先写完开头，再对照审核标准逐项检查缺失

### 高武/升级文开篇特征

**高武开头的期待感来源**
- 前期没有大爆点，但靠"练武→拜师→比赛"的升级线拉住读者
- 第一次以读者视角看能看下去，但拆解时觉得无聊——因为吸引力来自节奏而非剧情转折
- 这类书核心靠升级循环的稳定期待感，不靠单点爆点

**单元故事开篇的循环套路**
- 标准循环：展示优势→制造信息差→展示信息差→装逼震惊→收获奖励→铺垫下文
- 每个循环可以独立成章，循环之间用伏笔串联
- 单元故事内要保证"起→转→合"完整，哪怕只有3000字

### 主角悲惨经历的放置时机

- 没有固定答案，应从情绪角度分析实际问题
- 如果目标受众能共鸣（如小镇做题家刷题走出深山的艰辛），可以放开头
- 如果没有现实共鸣（如灭族），读者内心不会关心，放在后期以期待感形式慢慢揭晓
- 《夜无疆》《完美世界》等玄幻经典就是把悲惨过往后期揭晓的范例

## 开头常见问题与改进

### 开篇最常见致命问题

**信息传达不清**
- 主角在哪、干什么、动机、金手指、世界观——至少要让读者读懂基本设定
- 大量篇幅写无关内容（如无意义的打斗），对后续剧情毫无推动
- 出场过多无关人物，大段废话没有推进主线

**人设逼格过低**
- 主角看见美女就紧张——放配角有特点，放主角很low
- 过多无意义的内心戏描写，降智拉低逼格
- 动不动爆粗口，降低主角形象
- 主角一定要有b格，配角可以奇形怪状但要有特色

**情绪缺失**
- 最大问题往往不是文笔，而是没有情绪
- 战斗枯燥无味——核心在于战斗中情绪的刻画
- 情绪太淡，看几段就不想读下去

**节奏问题**
- 第三章节奏容易出问题：出场过多无关人物
- 与主线无关的互动需要删掉
- 铺垫太多迟迟不进入主线

### 改进方向

**强情绪开头**
- 从高潮事件切入（如发现自己没死）
- 或复仇类强情绪开头（主角家人受尽欺负到极致）
- 重大事件不能写得像闹着玩

**情报与期待感**
- 每个情报都要有期待价值，废情报直接删
- 关键情报要拉起期待——给读者一个"然后呢？"
- 可以通过情报先立人设

**目标驱动**
- 主角目标要交代清楚（想获得什么、打败谁）
- 危机开头的意义：要么主角战胜并收获，要么不写
- 打斗必须基于主角目标，让主角有收获

**与读者共鸣**
- 通过角色处境与读者受众产生共鸣
- 表明立场——先让读者站在主角这边
- 杂活累活还背锅的处境能引起打工人共鸣

### 角色与底线

**负面角色的使用**
- 可以存在但必须有不可替代的剧情功能
- 不能纯为恶心读者而设计
- 在没有强烈危机冲突的文里可以起到压抑情绪作用
- 主角足够强势时，任何角色都能变成正反馈

**内容红线**
- 擦边和三观不正的内容风险极大
- 如果非要写，用外国背景，提前跟编辑打招呼
- 被举报就完了，不要冒险

**女频特殊规则**
- 双洁是基础规则（处男处女+初恋+唯一白月光）
- 不要挑战此规则
- 补救方案：用"其实是同一个人（转世/穿越）"来化解

### 典型问题案例与修改建议

**案例1：信息传达不清**
- 症状：开局写主角和老虎打架，然后被秒杀，读者完全看不懂意义在哪
- 问题：主角在哪、干什么、动机、金手指、世界观一个都没交代清楚
- 修改：开局直接交代穿越背景和处境，明确主角当前目标再进入战斗

**案例2：金手指出场突兀**
- 症状：主角像写日记一样突然冒出一个金手指
- 问题：金手指出现缺乏触发条件，没有和情节自然结合
- 修改：用危机或困境触发金手指，让金手指成为解决问题的手段而非突然掉落

**案例3：期待感缺失**
- 症状：书名"天地同寿，但不消耗我的寿命"，读者看完不明所以
- 问题：金手指功能不明确，没有直接点出能干什么，读者无法联想期待
- 修改：书名/简介要具体——能穿越两界？能无限自爆？功能要说清楚

**案例4：简介东扯西塞没有主题**
- 症状：简介里各种设定堆砌，没有一个核心卖点
- 问题：简介像设定说明书而非广告，没有吸引点击的钩子
- 修改：简介只写三件事——主角处境、金手指能干嘛、第一个爽点预览

**案例5：战斗枯燥无味**
- 症状：大量打斗描写但读者无感，甚至跳过
- 问题：战斗没有情绪，纯粹写招式和数值变化
- 修改：战斗中必须穿插情绪刻画——恐惧、愤怒、决心、意外反转

**案例6：铺垫章掉追读**
- 症状：写铺垫/分析/博弈类章节名后追读暴跌
- 问题：章节名暴露了这是铺垫，读者直接跳过
- 修改：铺垫章也要用有戏剧性的标题把读者"骗"进来

### 开头情绪波动线（实操示范）

**情绪上行-下行交替法**
- 先给读者一个高起点：穿越成了仙人（情绪上行，觉得牛逼）
- 再拉下来：只是记名弟子，练气九层（情绪下行）
- 再拉更低：TM就是个劳务派遣，杂活累活还背锅（下行到最低）
- 共鸣点：与读者受众产生共鸣，表明立场
- 转机：有金手指了（上行）
- 再转折：金手指是自杀和敌人同归于尽的——用了敌人不一定死，自己肯定死（下行+戏剧性）
- 绝望：这和穿越前有什么区别（情绪触底）

**关键原则**
- 每次情绪波动都要有具体信息支撑，不能空转
- 共鸣是关键步骤——让读者站在主角这边
- 戏剧性情节必须写出来，不能只存在于设定中

### 金手指与主角人设匹配

**常见翻车：金手指与人设不契合**
- 冷酷型主角配搞笑金手指 = 违和
- 搞笑型主角配极端自爆金手指 = 不搭
- 金手指要与主角性格、世界观相契合

**正确匹配思路**
- 自爆类金手指适合开朗/搞笑型人设——装逼时有开挂爽感，吐槽时也契合
- 心狠手辣型主角适合传统型金手指（升级、资源获取类）
- 金手指功能如果和主角优势冲突，读者会觉得别扭

### 创意的正确展开方式

**有创意不等于写好了**
- 挂一个"倒霉"的名字，实际写的是普通积分类系统 = 空有皮没有内核
- 必须把创意点变成剧情写出来，用对比放大感觉

**用对比把创意写活**
- 例"倒霉"创意：同门都成弟子，主角被选为大药要练丹；正要杀他时紧急缺药材来不及洗涤
- 主角把霉运转移给炼丹师——对方功法出差错吐血，主角偷丹药逃跑
- 这才是把"倒霉"这个创意写出来了

**书名要与内容匹配**
- 书名写"苟"但主角性格张扬 = 跑题
- 书名写"倒霉"但内容是普通系统文 = 挂羊头卖狗肉
- 先想清楚核心创意是什么，书名、简介、正文三者统一

### 新人最容易被忽略的问题

**简介称人设出戏**
- 简介最后自称"俺"——太出戏，换成其他自称
- 人物说话要白话自然，不要刻意文言或书面语

**事件选择的戏剧性**
- 选择没有戏剧性的事件开局注定失败
- 参考对标书的开局事件——它有被验证过的趣味性
- 自创事件如果觉得无聊，大概率读者也觉得无聊

**多线扑街后的诊断思路**
- 连扑三本：先看是不是对标书没选对
- 分析同题材同平台成功作品的开头结构，提炼共性模式
- 看不明自己的问题时：把书名简介正文分开审视，问题往往在最基础的地方

## 黄金开篇技巧

- **开头视角选择**：摒弃古上的上帝视角开篇，以主角视角切入让读者迅速代入，建立"我在经历这件事"的沉浸感
- **开头要有具体事件**：避免世界观说明书式开头，第一件事就要有事情发生，让读者被事件带着走而非被设定淹没
- **三种开局方法**：危机开局（主角上来就面临生死压力）、悬疑开局（抛出悬念钩住好奇心）、反转开局（颠覆读者预期制造惊喜）
- **金手指出现时机因题材而异**：系统流第一章就必须出金手指，其他题材可延后——大奉打更人第10章才出金手指照样成功，关键是金手指出现前剧情本身要足够吸引人
- **开头节奏因题材而异**：不同题材的读者耐心不同，需结合大纲构架决定前几章的节奏松紧，不能一刀切

## 开局三大基点与核心模板

### 三大基点
所有成功的开局本质上都在做三件事，顺序和侧重不同：

| 基点 | 内容 | 作用 |
|------|------|------|
| 人设基点 | 展示主角的核心性格和处境 | 建立代入感和共情 |
| 切入点基点 | 主角遭遇的第一个冲突/机遇 | 制造紧迫感和阅读动力 |
| 金手指基点 | 展示主角的独特优势 | 制造期待感和差异化 |

### 两个核心开局模板
所有开局模板都是以下两个核心模板的变体：

**模板一：困境切入型**
1. 主角处于不利处境（人设基点 = 共情入口）
2. 遭遇压迫/危机（切入点基点 = 紧迫感）
3. 金手指激活/展示（金手指基点 = 反转期待）
4. 初步解决问题但引出更大问题（钩子）

**模板二：优势切入型**
1. 主角已有独特优势但被低估/隐藏（人设+金手指合一）
2. 优势在某个场景中意外暴露（切入点）
3. 周围人的认知被刷新（震惊反应）
4. 暴露引来更大的关注/麻烦（钩子）

### 基点组合原则
- 三个基点在前3章内必须全部完成
- 哪个基点放在第一章取决于题材：困境型先人设，优势型先金手指
- 切入点必须跟主线强相关，不能是随机事件

## 题材开头数据库

> 从大量热门网文的第一章提炼，按题材分类的开头写法参考。每个题材提供：真实开头范例 + 结构分析 + 可直接套用的模板。

### 都市脑洞

**范例**：《病娇妈妈们找上门，儿女都吓哭了》
**开头结构**：场景→独处→意外→反转→系统

> 江城市。破旧楼房内。苏牧下班后没回家。他跟老婆江亦瑶撒了个谎，说要出差。实际上，他躲到了这里。

**结构拆解**：
1. 2字交代地点（江城市）
2. 5字交代场景（破旧楼房内）
3. 一句话交代状态（下班没回家）
4. 一句话制造悬念（撒谎，躲）
5. 然后展开：意外女人出现→仙人跳→离婚→系统激活

**可套用模板**：
```
{城市名}。
{简短场景描述}。
{主角}{一个反常行为}。
{一句话解释为什么这么做，但留悬念}。
实际上，{真相，制造更大悬念}。
```

### 豪门总裁

**范例A**：《禁她入骨》
**开头结构**：极端环境→感官描写→身份交代→被卖真相

> 东南亚，帕孔地区。盛夏。烈日高悬，炙烤大地。空气又干又烫，每一口呼吸都如在吞咽含沙的热风。四周散发着尿液的骚臭味，汗液的酸腐味。

**结构拆解**：
1. 地点+季节（东南亚，盛夏）
2. 温度+嗅觉感官
3. 困境（铁笼，被当猪仔）
4. 主角身份（大二学生，被骗）
5. 背叛者揭晓（亲叔叔）

**可套用模板**：
```
{极端/异国地点}。{季节}。
{2-3个感官细节}。
{困境描写，1-2句}。
{主角身份交代}。
{一句话揭示谁害了她}。
```

**范例B**：《不愿再让你低头》
**开头结构**：医院→诊断结果→倒计时→电话

> 出了医院，浓重的夜色压下来，秋风卷着落叶吹来，刚从医院出来的岳绮颜下意识的拢了拢风衣，握着诊断书的手指因为用力骨节泛出青白色。

**结构拆解**：
1. 医院出口（暗示坏消息）
2. 天气描写映射心情
3. 握诊断书的动作（不说内容，制造悬念）
4. 医生话引用（最多两年半）
5. 打电话给丈夫（核心关系）

**可套用模板**：
```
出了{医院/法院/某场所}，{天气映射心情}。
{主角}{一个微动作暗示内心}。
脑海中闪过{某人}的话——
"{冲击性的一句话}"
{主角的反应}。
{一个动作推动下一步}。
```

### 古代宅斗

**开头结构**：核心事件→被弃→回归→反差

> 债主把我和妹妹卖去西域青楼抵债。爹娘凑到的钱只够赎一人，他们毫不犹豫的救了妹妹。

**结构拆解**：
1. 一句话核心设定（被卖）
2. 父母选择（救妹妹不救我）
3. 十年后对比（妹妹入东宫 vs 我成摄政王）
4. 回归场景（走正门）
5. 父母反应（嫌弃→恐惧）

**可套用模板**：
```
{N年前，被害/被弃的事件}。
{亲人选择救别人不救主角}。
"{被弃时的伤人台词}"
{N年后，主角已{隐藏身份/实力}}。

```

### 玄幻修仙

**范例A**：《西游你叫我吗喽》
**开头结构**：世界危机→日常切入→反差主角→系统/穿越

> "听说了吗，樱花国京都，被一只青毛巨狮占据……"

**结构拆解**：
1. 群众对话交代世界观
2. 危机背景（西游妖魔入侵现实）
3. 学校场景（日常反差）
4. 主角报名（所有人觉得他疯了）
5. 穿越者优势（知道西游记剧情）

**可套用模板**：
```
"{一条震惊的新闻/传闻}"
"{群众反应/讨论}"
——安静——
{场景转换：学校/公司/日常}
{权威人物宣布一个危险任务}
{所有人退缩}
"我报名！"
{主角站出来，所有人震惊}
{暗示主角有隐藏优势}
```

**范例B**：《长生火影》（动漫衍生）
**开头结构**：原作角色互动→穿越者内心→系统选择

**可套用模板**：
```

### 青春校园

**范例**：《你携盛夏，渡我寒冬》
**开头结构**：苦难叠加→遇见→改变

> 最走投无路那年，我遇到了林燃。我被同学欺负，被亲妈揍，被邻居们骂野种。

**结构拆解**：
1. 倒叙钩子（最走投无路那年）
2. 三重苦难叠加（欺负+家暴+歧视）
3. 遇见（他/她出现）
4. 第一印象（不良少年形象）
5. 交易（我帮你写作业，你保护我）

**可套用模板**：

```

### 现实情感

**范例**：《五年恩情，一朝两清》
**开头结构**：纪念日被放鸽子→揭穿→回忆契约

> 顾泽言的白月光林薇回国那天，是我们结婚三周年纪念日。我坐在餐厅靠窗的位置，看着桌上冷透的牛排和燃尽的蜡烛。

**结构拆解**：
1. 两个事件叠加（纪念日 vs 白月光回国）
2. 场景细节（冷牛排、燃尽蜡烛=被遗忘）
3. 手机消息（"临时有急事"）
4. 朋友圈验证（白月光发图）
5. 契约婚姻背景交代

**可套用模板**：
```

{被忽视的场景细节，暗示等待}。

{主角查看{朋友圈/聊天}——真相}。
{一句话内心独白，冷静但心碎}。
```

### 点众轻喜剧

**范例**：《捡回来的小乞丐，竟然想咬我》
**开头结构**：吐槽→捡人→反差→笑料→升级

> 许寂看着雪地里那个浑身是血、眼神凶得像要吃人的小罗莉，叹了口气。"这年头，碰瓷的成本都这么高了吗？"

**结构拆解**：
1. 主角吐槽式反应（不是害怕是吐槽）
2. 建立世界观（修仙+穿越10年+没金手指）
3. 捡到危险生物（小萝莉=鬼帝重生）
4. 认知错位制造笑点（鬼帝咬不动凡人）
5. 一碗粥引发升级（种田天赋+灵气）

**可套用模板**：
```
{主角}看着{危险/奇怪的事物}，{非正常反应}。
"{吐槽台词}"
{世界观一句话交代：穿越N年/系统/金手指状态}
{捡到/遇到一个人/物}
{对方的真实身份，与表面反差极大}
{对方的攻击完全无效→制造笑点}
{主角毫无危机感地照顾对方}
{意外发现对方获得好处→双赢设定}
```

### 开头通用规则

**基本要求**：
- 第一个一级结构中，主要角色（主角+可攻略对象+主要配角+主要反派）不得登场3个以上
- 开头必须以叙事为主，压缩介绍性、抒情性和环境描写文字
- 叙事优先于塑造人设，人设尽量通过叙事手法体现
- 叙事连贯：正叙为主，不用插叙，不切换视角，不用回忆/梦境/大篇幅对话
- 绝不水字数——对话是水字数重灾区，开头两章不写介绍信息的对话，只写推进剧情的对话
- 第一章不上来就写打斗（小说需要大量信息铺垫才能在打斗中产生爽感）
- 300字内主角必须登场，1000字内必须出现爽点或期待点
- 不写序章/楔子/引（除非叙事为主、足够简短且与第一章有密切逻辑联系）
- 第一章由主角行动引导故事走向，主角不能"打酱油"
- 世界观详细解说至少等到第一个一级结构结束后
- 爽文第一章内表明主角有装逼的能力或潜力

**开局环境要求**：
- 第一个环境应无需动脑、无需过多设定解释就能理解
- 最好有一个方便多角色出场的环境，便于调动人物、发展剧情

**开头布局要求**：
- 氛围应轻松明快或平顺自然，切忌开头悲伤压抑
- 切入点（触发点）应在第一个一级结构中出现，最好在第一章
- 切入点是叙事主角人生的转折点，之后主角进入非常态生活
- 情绪安排：尽快迎来第一次上行。推荐"先短暂下行再上行"或"原地起飞"
- 第一个冲突应影响重大，事先点明解决的好处和不解决的坏处
- 主角行动必须带来正面收益，最好是主动选择使用"神器"解决问题
- 第一章必须说明主角当下的目标（短期目标要紧迫，长期目标要引人期待）
- 第一章必须写明本文的卖点

**开头期待感三路径**：
- 事业线突破口：设定/素材新颖，读者预见喜闻乐见结局但猜不到过程
- 爱情线突破口：人设或人设搭配新颖有吸引力，开头就抛出亮点
- 双线缠绕突破口：事业线与爱情线在第一个一级结构同时进行，互相影响

**开头噱头类型**：
- **事件噱头**：以一个事件快速进入故事（常见于飞卢）
- **金手指噱头**：先描写主角困境，引出目标，金手指第一次施展
- **人设噱头**：不建议新手使用，出彩人设极少
- 事件噱头和金手指噱头的写法不同，不能混用
- 噱头吸引读者后，立刻嫁接主线（主角的目标），事件结束后必须马上接上新目标

**噱头三分法详解**：

*事件噱头写法*：
- 书名即事件，如"账号被找回，我反手充值一百万"
- 事件作为切入点，极快进入故事，通常写五章左右
- 强烈建议用日常常见事件开局（天然有代入感）
- 一上来就进入事件，不要铺垫穿越/绑定金手指
- 错误示范：用金手指写法去写事件（读者冲着事件进来，你磨磨唧唧写主角和困境）

*金手指噱头写法*：
- 金手指分布在全文，前期出现多，后期逐渐减少
- 标准流程：描写主角 → 当前困境 → 引出目标 → 金手指第一次施展 → 取得少许成果
- 金手指可以结合事件写法：开局事件安排在金手指出现前，事件拉成主线，由主线诞生目标，再出现金手指

*人设噱头写法*（仅限高阶）：
- 出彩人设能直接影响剧情构建（如慎重勇者，性格决定阶段剧情发展）
- 杀伐果断/冷酷无情只是当下基本要求，不算出彩人设
- 人设噱头多出现在日常文、恋爱文，由人设引发戏剧性
- 新人不推荐：人设写不对就成了深井冰，读者看不懂

**七步法第一章字数分配参考**（《从斩妖除魔开始长生不死》）：

| 段落 | 字数 | 内容 |
|------|------|------|
| 世界观+穿越事实+前身经历 | 300字 | 快速交代背景，不啰嗦 |
| 第一个吸睛小剧情 | 500字 | 用事件制造初始吸引 |
| 前置金手指获得安全感 | 400字 | 金手指首次展示 |
| 感受到不安全 | 400字 | 新的威胁出现 |
| 遇见危机 | 500字 | 核心冲突到来 |
| 直接使用金手指 | 400字 | 解决问题，完成第一个爽点 |
